# User Guide - Web Portal Informasi Perusahaan
*(Draft/Kerangka Dasar)*

Dokumen ini adalah panduan pengguna akhir (Administrator/Staff Publikasi) dalam mengelola Web Portal Informasi Perusahaan.

## 1. Pengenalan Web Portal
- **Fungsi:** Menampilkan profil perusahaan, layanan, portofolio, dan artikel publik.
- **Akses Publik:** `https://[domain-perusahaan].com`
- **Akses Admin:** `https://[domain-perusahaan].com/admin` (contoh)

## 2. Manajemen Konten (Admin Dashboard)
- **Login:** Masuk menggunakan kredensial admin yang diberikan tim IT.
- **Mengelola Halaman (Pages):** Cara mengedit teks pada *Company Profile* atau *Landing Page*.
- **Mempublikasikan Artikel/Berita:** Cara membuat post baru, mengunggah gambar *thumbnail*, dan mengatur kategori berita.

## 3. Integrasi Layanan (Pintasan Eksternal)
- Karyawan dapat menemukan tombol atau menu *Login ERP* dan *Login Asset Management* di bagian footer atau navigation bar (khusus portal internal). Menu ini akan mengarahkan user ke sistem ERPNext (`erp.domain.com`).
