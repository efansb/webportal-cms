# SURAT PENAWARAN BIAYA (RAB)
**Pengembangan Infrastruktur IT Terpadu, ERP, dan Portal Web Perusahaan**

## 1. LINGKUP PEKERJAAN
*   **Fase 1: Infrastruktur Server:** Persiapan, optimasi, dan konfigurasi Proxmox Home Server (LXC/VM, Reverse Proxy, Tunneling/Networking, Keamanan).
*   **Fase 2: Sistem Inti (ERP & Asset Management):** Instalasi, penyesuaian dasar, dan implementasi ERPNext (HR/Operasional) serta openMAINT (Manajemen Aset).
*   **Fase 3: Pengembangan Aplikasi Kustom:** Desain UI/UX VVIP, pengembangan Frontend & Backend Web Portal Informasi Perusahaan (Custom).
*   **Fase 4: Training & Deployment:** Migrasi data, simulasi (*User Acceptance Test*), Training, Peluncuran (Go-Live), hingga garansi/maintenance.
